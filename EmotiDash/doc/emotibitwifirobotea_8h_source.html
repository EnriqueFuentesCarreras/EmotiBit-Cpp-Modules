32/**
33 * @file emotibitwifirobotea.h
34 * @author Enrique Fuentes
35 * @date 2024-2025
36 * @brief Clase para gestionar la comunicación WiFi con dispositivos EmotiBit integrada en RoboTEA.
37 *
38 * Esta clase proporciona métodos y estructuras para establecer y manejar conexiones WiFi
39 * con dispositivos EmotiBit, encargándose de aspectos como publicidad en la red,
40 * intercambio de datos, control de grabación, sincronización de tiempos y gestión de conexiones.
41 *
42 * @details
43 * - Gestiona sockets UDP y TCP para datos y control.
44 * - Implementa mecanismos para publicidad y descubrimiento automático de dispositivos.
45 * - Ofrece funcionalidades para el envío y recepción de paquetes de datos y control.
46 * - Utiliza multihilo para el procesamiento asíncrono y simultáneo de datos y publicidad.
47 * - Proporciona interfaces para integración en aplicaciones Qt.
48 *
49 * @note Este código ha sido adaptado y refactorizado por Enrique Fuentes para integrarse en RoboTEA.
50 * Originalmente basado en el proyecto EmotiBit WiFi Host desarrollado por Connected Future Labs.
51 *
52* @author Enrique Fuentes
53* @date Mayo 2025
54*
55* @details
56* Proyecto desarrollado en C++ con Qt 6.7.2 para la UNED.
57*
58* - Framework: Qt (módulos: Network, Widgets, Core, etc.)
59* - Compilador: MSVC / MinGW (según configuración)
60* - Entorno: Windows 10/11
61* - Herramientas de documentación: Doxygen, Graphviz
